On Nunjucks

Nunjucks is a rich and powerful templating language from mozilla.

For your prototype to be rapid and easy to update, it pays to modularize your repeateable code in partials or templates.

Using Templates

  1. create a template on

    app/templates

  2. create variables with

    block

  3. create pages on

    app/pages

  4. assign layout with

    extend

  5. add variable contents with

    block

  6. results (template + page) appear on

    dist

Using Partials

  1. create a partial on

    app/templates/partials

  2. call it on

    app/templates

    or

    app/pages

    with

    include

  • explain macros (creation and calling)
